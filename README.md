## simple node.js back end for expense tracker app
### Features
- **API Endpoints:**
  - income/budget API
  - expense API
    
- **User Management:**
  - Role-based access control (Admin, User)
  - Secure authentication using JWT (JSON Web Tokens)
    
####
Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
